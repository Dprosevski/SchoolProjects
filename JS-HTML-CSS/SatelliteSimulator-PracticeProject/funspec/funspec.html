Satellite Simulator
By Dimitrije Prosevski



  • Purpose

    The main purpose of the project "Satellite Simulator" will be to demonstrate the behavior of the satellite under different conditions. The project is planned to perform all the physics calculations for desired satellite around any planet from the Solar System that user chooses. Each satellite on a different planet will be affected differently because of the unique conditions of each planet. Detailed steps of computations will be provided for the user to see/learn.

  • Objectives

    The objective of the project will bo to allow user to input mass and altitude of the satellite. From there, the code will automatically calculate:
    • Surface Weight
    • Orbital Radius
    • Gravitational Force
    • Orbital Velocity
    • Escape Velocity
    • Acceleration
    • Angular Momentum
    • Period

  • Primary Audience

    The website's primary audience would be college students or any adults who are interested in learning about the behavior of the satellites and physics behind them for academic/research purposes.
